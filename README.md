## ğŸš€ Quick Start

1. **Clone the repository**
   ```bash

   cd movie-catalog
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

   ```env
   VITE_TMDB_ACCESS_TOKEN=your_tmdb_access_token_here
   ```

   ```bash
   npm run dev
   ```

   ```bash
   npm run build
   ```
